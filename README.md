# DIY Signal Hunting Lab 📡

A homemade RF experimentation project built using ESP32, DIY copper-wire antennas, improvised signal extenders, HexPulse firmware, and HexMind AI.

This project demonstrates how low-cost electronics and simple handmade antennas can be used for:
- Wi-Fi signal testing
- Bluetooth signal scanning
- RF experimentation
- DIY electronics learning
- Portable field testing

Built completely with homemade components, portable batteries, copper wire antennas, and open-source tools.

# Homemade Antenna Design 📡

This setup uses a very simple homemade antenna system inspired by:
- Directional Wi-Fi reflectors
- Basic Yagi concepts
- RF field testing setups
- Portable DIY signal hunting tools

The antenna is made using:
- Copper wires
- Aluminum foil reflector
- Lightweight frame
- Manual wire alignment

The goal is not maximum gain, but simple experimental learning and testing.

# DIY Build Guide 🛠️

## Step 1 — Create Reflector

Use:
- Cardboard
- Aluminum foil

Wrap the cardboard with foil to create a directional reflector.

---

## Step 2 — Build Wire Antenna

Using copper wire:
- Create crossed elements
- Leave spacing between sections
- Secure with tape or glue

---

## Step 3 — Connect to ESP32

Attach the antenna carefully to the ESP32 RF section.

---

## Step 4 — Portable Power

Use:
- Power bank
- Lithium battery
- USB cable

This allows outdoor field testing.

---

## Step 5 — Start Testing

Launch HexPulse firmware and begin signal testing.
